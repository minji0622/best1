💡 엑셀 수식 검증 가이드

대시보드 수치가 맞는지 엑셀에서 직접 확인해보세요. 아래 수식을 복사하여 엑셀 시트의 빈 셀에 붙여넣으세요.

* 원본 데이터 시트의 컬럼 위치를 기준으로 작성되었습니다. (A:주문일시, B:회원구분, C:결제금액, D:주문횟수 가정)

1. 월간 총 매출액 확인

=SUMPRODUCT((TEXT(A:A,"YYYY-MM")="2026-02")*(C:C))

2026년 2월의 모든 결제금액(C열)을 합산합니다.

2. 월간 신규 매출액 확인

=SUMPRODUCT((TEXT(A:A,"YYYY-MM")="2026-02")*(D:D<=1)*(C:C))

2026년 2월 중 주문횟수(D열)가 1 이하인 금액만 합산합니다.

3. 월간 활동 회원수 (중복제거)

=ROWS(UNIQUE(FILTER(E:E, TEXT(A:A,"YYYY-MM")="2026-02")))

주문자ID(E열)를 기준으로 중복을 제외한 명수를 구합니다. (Excel 365용)

4. 특정 회원유형 매출 확인

=SUMIFS(C:C, A:A, ">=2026-02-01", A:A, "<=2026-02-28", B:B, "기업회원")

특정 날짜 범위 내의 '기업회원' 매출액만 합산합니다.
